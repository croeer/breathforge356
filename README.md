# Everbreathe

A breathing training app based on the 356 method - 3 sessions per day, 6 breaths per minute, 5 minutes per session.

## Features

- Breathing animation: 4 seconds inhale, 6 seconds exhale
- Session tracking: Track 3 sessions per day (5 minutes each)
- Daily reminders: Get notified 3 times a day to practice
- Beautiful bluish color scheme inspired by breathing

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:4321` to see the app.

## Build

```bash
npm run build
```
